# Simulador de Surtidor de Nafta con Interfaz Gráfica

Este es un proyecto que desarrollé para la facultad en Java. Es un software interactivo que simula la atención de clientes en una estación de servicio.

El sistema gestiona variables como la capacidad máxima del tanque, la cantidad inicial de combustible, el precio por litro, el tiempo de inactividad, los litros entregados y el dinero recaudado en cada transacción.

### Utilize para el desarrollo de este codigo

**Lenguaje:** Java
**Librerías Gráficas:** Java Swing / AWT (JFrame, JButton, JTextArea)
**Entorno de Desarrollo:** IntelliJ IDEA
